Tampermonkey / Violentmonkey Helper Script

Because of browser security rules (Same-Origin Policy), NetSanctum cannot read the URL or cookies/tokens from the iframe directly. Install this small userscript in Tampermonkey or Violentmonkey. It automatically detects when you browse any Lib network page inside the iframe, extracts the authorization token, and fills it into NetSanctum in real-time.

Install Automatically / Установить авто Get Extension: Chrome Firefox
Если авто-установка не сработала / If auto-install fails:
Из-за политик безопасности браузеров на localhost расширение может заблокировать фоновую загрузку скрипта. Скопируйте код ниже кнопкой "Copy" и вставьте вручную: откройте панель Tampermonkey -> Создать новый скрипт -> вставьте код -> Файл -> Сохранить. 
// ==UserScript==
// @name         NetSanctum Autofill Helper
// @namespace    http://tampermonkey.net/
// @version      0.1
// @description  Autofills Lib network titles/tokens to NetSanctum
// @match        https://*.mangalib.me/*
// @match        https://*.ranobelib.me/*
// @match        https://*.hentailib.org/*
// @match        https://*.slashlib.me/*
// @match        https://*.comixlib.me/*
// @match        https://*.anilib.me/*
// @grant        none
// ==UserScript==

(function() {
    'use strict';
    if (window.self !== window.top) {
        function sendUpdate() {
            let token = localStorage.getItem('token') || localStorage.getItem('authorization');
            if (!token) {
                for (let i = 0; i < localStorage.length; i++) {
                    let key = localStorage.key(i);
                    if (key && key.toLowerCase().includes('token')) {
                        let val = localStorage.getItem(key);
                        if (val && val.length > 20) { token = val; break; }
                    }
                }
            }
            window.parent.postMessage({
                type: 'netsanctum-nav',
                url: window.location.href,
                token: token
            }, '*');
        }

        sendUpdate();
        let lastUrl = window.location.href;
        new MutationObserver(() => {
            if (window.location.href !== lastUrl) {
                lastUrl = window.location.href;
                sendUpdate();
            }
        }).observe(document, {subtree: true, childList: true});
    }
})();

FAQ / Справка

Руководство по использованию модуля скачивания и интеграции с Lib Network

1. Как качать тайтлы / How to Download

Чтобы поставить тайтл на закачку, перейдите на вкладку "Скачать" (или "Найти") и введите ссылку на главную страницу произведения из сети сайтов Lib Network (MangaLib, RanobeLib, AniLib и др.).

Пример рабочей ссылки: https://mangalib.me/manga/chainsaw-man

Для скачивания манги, ранобэ или аниме нажмите кнопку скачивания. Задача отправится в Celery воркер, а прогресс скачивания отобразится в правой колонке вкладки "Скачать".

2. Вкладка "Найти" и Помощник / Find Tab & Autofill

Вкладка "Найти" содержит встроенный фрейм с ресурсами. Чтобы автоматизировать перенос ссылок и авторизацию из фрейма, мы создали вспомогательный скрипт.

Вы можете установить скрипт в один клик, если у вас уже установлена библиотека расширений (Tampermonkey для Chrome или для Firefox). После установки скрипта, при переходе на любую страницу произведения внутри фрейма, его URL и токен авторизации будут автоматически заполняться на вкладке NetSanctum.

Примечание: Если автоматическая установка на localhost блокируется политиками безопасности расширения, скопируйте код скрипта вручную из панели "Autofill Helper" на вкладке "Найти" (кнопка копирования и код доступны там).

3. Форматы файлов и Чтение / File Formats & Sync

Все скачанные материалы хранятся локально в оптимизированных форматах:

  • Манга/Хентай: Сохраняется в архивах .cbz. Читать можно через наш встроенный премиум-ридер во вкладке "Библиотека".
  • Ранобэ/Книги: Сохраняется в формате .epub со всеми иллюстрациями.
  • Аниме: Видеофайлы скачиваются в исходном качестве (.mp4/.mkv).

Также доступен экспорт на ПК или синхронизация с мобильным приложением NetOutpost (Android).

4. Авторизация и Ограничения / Auth & Limits

Для скачивания некоторых глав, имеющих возрастные ограничения или ранний платный доступ, необходимо быть авторизованным на целевом сайте.

Важно: Вход через Google/Yandex внутри iframe заблокирован провайдерами OAuth. Для авторизации воспользуйтесь кнопкой "Open in New Tab", зайдите в свой профиль, скопируйте ваш токен авторизации и укажите его глобально во вкладке "Настройки" или вставьте в форму скачивания.
